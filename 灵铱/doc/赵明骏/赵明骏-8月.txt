        8.7号：
            学习scrapy对接docker
            scrapy对接Docker
 
            测试网站：http://quotes.toscrape.com/
            git clone git@github.com:Germey/ScrapyTutorial.git
             
            首先在项目的根目录创建requirement.txt 文件将整个项目依赖的python全部列出来
            scrapy
            pymongo
            也可以写上版本号：
            scrapy>=1.4.0
            pymongo>=3.4.0
             
            在项目的根目录创建Dockerfile，不加任何后缀：编写内容如下：
            FROM python:3.5
            ENV PATH /usr/local/bin:$PATH
            ADD . /code
            WORKDIR /code
            RUN pip3 install -r requirement.txt
            CMD scrapy crawl quotes
             
            2、修改数据库配置，修改ip
             
            3、构建镜像，执行命令：在ubuntu中测试，需要加sudo
            sudo docker bulid -t quotes:latest .
             
            执行过程中输出如下（部分）：
            Status: Downloaded newer image for python:3.5
             ---> 049768b1d61a
            Step 2/6 : ENV PATH /usr/local/bin:$PATH
             ---> Running in ae56702af0b5
            Removing intermediate container ae56702af0b5
             ---> 06ec7160025f
            Step 3/6 : ADD . /code
             ---> 3f9d6dfd6200
            Step 4/6 : WORKDIR /code
            Removing intermediate container d63bd9c7e3c5
             ---> efcc64115375
              
             4、查看构建的镜像：
            docker images
             
            返回的结果：
            quotes latest 41c8499ce210 2 miuntes ago 769 MB
             
            5、镜像可以在本地测试运行，执行的命令如下：
            docker run quotes
             
            6、将项目推送带Docker Hub 这样就可以从远程服务器下拉镜像运行
             
            在https://hub.docker.com/新建一个账号，并且新建一个Repository,名称为quotes，比如我的用户名为zmj，新建的Repository名称为quotes,那么Repository，
            的地址就可以用germey/quotes
             
            为新建的镜像打一个标签，命令是：
            docker tag quotes:latest germey/quotes:latest
             
            推送镜像到Docker Hub 命令是：
            docker push zmj/quotes
             
              
            7、如果我们想在其他的主机上运行这个镜像，主机上转好Docker，就可以执行下面的命令执行了：
            docker run zmj/quotes
